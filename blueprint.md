# Lotto Drawing App

## Overview

This is a simple web application that simulates a lottery drawing. It generates 6 unique random numbers between 1 and 45 and displays them to the user.

## Features

*   Generates 6 unique random numbers.
*   Displays the numbers in a visually appealing way.
*   Allows the user to trigger a new drawing with a button.
